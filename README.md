# [nocp.space](https://nocp.space)

This is Noah's Website written in Nextjs + Typescript.

## Deploy

```bash
git clone https://github.com/nocpiun/nocp.space.git
cd nocp.space
npm i
npm run build
npm run start # http://localhost:3000
```

## Development

```bash
npm run dev # http://localhost:3000
```
